# .github

predun 조직의 공용 설정 및 에셋을 관리하는 저장소입니다.

## 구조

```
.github/
├── profile/
│   └── README.md            # 조직 프로필 README (GitHub 조직 페이지에 표시)
├── assets/
│   ├── avatar.svg           # 조직 아바타 (심볼, 512×512)
│   ├── avatar.png           # 조직 아바타 PNG (GitHub 프로필용)
│   ├── logo-full-black.svg  # 라이트 모드용 풀 로고
│   ├── logo-full-white.svg  # 다크 모드용 풀 로고
│   └── brand-guide.html     # 브랜드 가이드 (전체)
└── README.md                # 이 파일
```

## 용도

- **`profile/`** — GitHub 조직 페이지에 표시되는 프로필 README
- **`assets/`** — 로고, 브랜드 가이드 등 조직 공용 에셋
